# Идентификаторы
1. Состоят из букв, цифр, '_'
2. Начинаются с буквы или '_'
3. Чувствительны к регистру

# Типы данных в cpp 
1. Целые числа:
    - `short(int)`
    - `int`
    - `long(int)`
    - `long long(int)` 
2. Символьный: char
3. Вещественные числа:
    - `float`
    - `double`
    - `long double`
4. Логический: bool
5. Специальные типы:
    - `nullptr_t`
    - `void`

# Модификаторы:
1. `signed/unsigned` - только для `char`, `int`, `long long`
2. `short` - для `int`
3. `long` - для `double`, `int`

! Формально, `long double` и `long long (int)` -
это отдельные типы данных, а не модификаторы для `double` и `int`. `long int` и  `short int` - все еще модификаторы!

# Размеры типов
1. В c++ типы не имеют фиксированной длины. В стандарте определено лишь неравенство.
```cpp
sizeof(short) ≤ sizeof(int) ≤ sizeof(long) ≤ sizeof(long long)
sizeof(float) ≤ sizeof(double) ≤ sizeof(long double)
```
2. В библиотеке [`<cstdint>`](https://en.cppreference.com/w/cpp/header/cstdint.html) определены типы с фиксированной точностью:
- int8_t, int16_t, int32_t, int64_t - знаковые целые числа
- uint8_t, uint16_t, uint32_t, uint64_t - беззнаковые
3. Узнать количество байт, можно воспользоваться встроенным в язык оператором `sizeof(<var>)`. Библиотека ['numeric_limits'](https://en.cppreference.com/w/cpp/types/numeric_limits.html), которая определена в хедере `limits` (`#include <limits>`)позволяет узнать больше информации о типе, к примеру минимум, максимум, наименьшее положительное значение(для вещественных типов).

# Литералы
## Целые числа
1. Постфиксы:
    - l приводит число к `long`
    - ll приводит число к `long long`
2. Префиксы
    - `0b` к двоичной системе счисления
    - `0` к восьмеричной
    - `0x` к шестнадцатерично
## Вещественные числа
1. Вариации литералов вещественных чисел
    - `0.12` - `double`
    - `0.12f` - `float`
    - `0.12l` - `long double`
    - `12e7l` - `long double`, но в экспоненциальной форме
2. Постфиксы:
    - l приводит число к `long double`
    - f приводит число к `float`
## Символьные 
1. Символьные литералы указываются в одинарных кавычках
2. Могут включать в себя escape-последовательности:
    - `'\t'` - таб
    - `'\\'` - бекслеш
    - `'\n'` - перевод строки 
    - `'\''` - одинарная ковычка
## Строковые
1. Строка - массив `char`
2. Любая строка заканчивается символом `'\0'`
# Преобразования типов
## Неявные
1. Сужающие, к примеру, `int a = 1.2f` -> дробная часть отбрасывается, возможное предупреждение о потере точности.
2. Расширяющие - компилятор приведет результат всего выражения к наиболее общему типу
## Явные
1. Применяются с помощью кастов:
```cpp
int a = (int)1.2f;             // -> 1
int b = static_cast<int>(2.7); // -> 2
```
2. Позволяют явно указать, к какому типу нужно привести значение, даже если происходит сужение или потеря данных.
