# Числовые последовательности

> [!Info]
> **Числовая последовательность** (далее — ч.п.) — это функция, которая множеству натуральных чисел ставит в соответствие множество вещественных чисел и обозначается $\{x_n\}: x_n = f(n), n \in N$, где:
> - $x_1, x_2, x_3, \dots x_n, \dots$ — элементы числовой последовательности
> - $x_n$ — общий член числовой последовательности

### Классификация последовательностей

| Название          | Определение                                          |
| ----------------- | ---------------------------------------------------- |
| Ограниченные      | $\exists M>0:\forall n \in N: \vert x_n\vert \leq M$ |
| Неограниченные    | $\forall M > 0: \exists n \in N: \vert x_n\vert > M$ |
| Возрастающие      | $\forall n \in N: x_n < x_{n + 1}$                   |
| Убывающие         | $\forall n \in N: x_n > x_{n + 1}$                   |
| Невозрастающие    | $\forall n \in N: x_n \leq x_{n + 1}$                |
| Неубывающие       | $\forall n \in N: x_n \geq x_{n + 1}$                |
| Строго монотонные | Возрастающие и убывающие                             |
| Монотонные        | Неубывающие и невозрастающие                         |


> [!Info]
> Числовая последовательность $\{x_n\}$ называется **бесконечно малой** (далее — б.м.), если $\forall \varepsilon > 0: \exists N(\varepsilon): \forall n > N(\varepsilon): |x_n| < \varepsilon$.

==Свойства бесконечно малых последовательностей==:
1. сумма и разность скольки угодно б.м. ч.п. есть б.м. ч.п.;
2. произведение скольки угодно б.м. ч.п. есть б.м. ч.п.;
3. произведение постоянной величины на б.м. ч.п. есть б.м. ч.п..

> [!Info]
> Числовая последовательность $x\{n\}$ называется **бесконечно большой** (далее — б.б.), если $\forall M > 0: \exists N(M): \forall n > N(M): |x_n| > M$

> **Теорема (1)** о связи б.м. и б.б. ч.п.
> 
> Если $\{x_n\}$, где $x_n \neq 0$, является б.б. ч.п., то обратная величина $\{\frac{1}{x_n}\}$ будет являться б.м. ч.п..

> **Теорема (2)**, обратная к т. (1)
> 
> Если $\{y_n\}$, где $y_n \neq 0$, является б.м. ч.п., то обратная величина $\{\frac{1}{y_n}\}$ будет являться б.б. ч.п..

==Свойства бесконечно больших последовательностей==:
1. сумма двух б.б. ч.п. одного знака есть б.б. ч.п. того же знака, что и слагаемые;
2. произведение двух б.б. ч.п. есть б.б. ч.п.;
3. произведений постоянной величины на б.б. ч.п. есть б.б. ч.п..

# Предел числовой последовательности

> [!Определение]
> Число $a \in R$ называется **пределом** числовой последовательности, если $\forall \varepsilon > 0: \exists N = N(\varepsilon): \forall n > N: |x_n - a) < \varepsilon$, и обозначается $\lim\limits_{n \to\infty}\{x_n\} = a$

> [!Определение]
> Последовательность, для которой существует предел $a \in R$, называется **сходящейся** в точке $a$.

*Замечание 1*: добавление или исключение из последовательности конечного числа её членов не влияет на её сходимость.
*Замечание 2*: если $\{x_n\}$ — сходящаяся последовательность, т.е. $\lim\limits_{n\to\infty}\{x_n\} = a$, то имеет место равенство $x_n = a + \alpha_n$. где $\alpha_n$ — общий член бесконечно малой ч.п. $\{\alpha_n\}$.
*Замечание 3*: 
- если $\{x_n\}$ — б.м. ч.п., то $\lim\limits_{n\to\infty}\{x_n\} = 0$, и наоборот;
- если $\{x_n\}$ — б.б. ч.п., все элементы которой имеют один знак, то $\lim\limits_{n\to\infty}\{x_n\} = \pm \infty$, и наоборот.

> **Теорема (3)** о "сжатой" последовательности
> 
> Если $\lim\limits_{n\to\infty}\{x_n\} = a$, $\lim\limits_{n\to\infty}\{y_n\} = a$ и $\exists N: \forall n \geq N: x_n \leq z_n \leq y_n$, то $\lim\limits_{n\to\infty}\{z_n\} = a$.
> 
> **Доказательство**
> 
> $x_n \to a, y_n \to a$ при $n \to \infty$; тогда $$\forall \varepsilon > 0: \exists N_1, N_2: \forall n > N_1: \vert x_n-a\vert < \varepsilon, \forall n > N_2: \vert y_n-a\vert < \varepsilon$$
> Пусть $N = \max\{N_1, N_2\}$, тогда $\forall n > N: a-\varepsilon < x_n, y_n < a + \varepsilon$.
> Учитывая, что $x_n \leq z_n \leq y_n$, имеем $a - \varepsilon < x_n \leq z_n \leq y_n < a + \varepsilon$, т.е. $a$ удовлетворяет определению предела для $\{z_n\}$, что и требовалось доказать.

> 
> Ограниченная и монотонная последовательность есть последовательность сходящаяся.

> **Теорема (5)**
> 
> Сходящаяся числовая последовательность имеет *один* предел.
> 
> **Доказательство**
> 
> Допустим, что $x_n \to a$ и $x_n \to b$, при $n \to \infty$ причём $a \neq b$. 
> Тогда $x_n = a + \alpha_n = b + \beta_n$, где $\alpha_n \to 0, \beta_n \to 0$ при $n \to \infty$.
> Представим разность $x_n - x_n = 0$ как $(a + \alpha_n) - (b + \beta_n) = (a - b) + (\alpha_n - \beta_n) = 0$. Отсюда $a - b = \beta_n - \alpha_n$, а так как $a \neq b$, то $\beta_n - \alpha_n \neq 0$, то есть общий член б.м. ч.п. $\{\beta_n - \alpha_n\}$ равен постоянной величине, отличной от нуля. Это противоречит определению б.м. ч.п..
> Значит, числовая последовательность не может иметь более одного предела, что и требовалось доказать.

> **Теорема (6)**
> 
> Если последовательность $\{x_n\}$ сходится в точке $a$, последовательность $\{y_n\}$ сходится в точке $b$ и при этом $x_n < y_n$, то имеет место неравенство $a \leq b$.
> 
> **Доказательство**
> 
> Допустим, что $x_n \to a$, $y_n \to b$ при $n\to\infty$, причём $a > b$. 
> Тогда $x_n = a + \alpha_n, y_n = b + \beta_n$, где $\alpha_n \to 0, \beta_n \to 0$ при $n \to \infty$.
> Согласно предположению теоремы, $x_n < y_n$, то есть $a + \alpha_n < b + \beta_n$.  Отсюда $a - b < \beta_n - \alpha_n$, тогда поскольку $a > b$, то $a - b > 0$, соответственно $0 < a - b < \beta_n - \alpha_n$, то есть общий член б.м. ч.п. $\{\beta_n - \alpha_n\}$ всегда больше постоянной величины, отличной от нуля. Это противоречит определению б.м. ч.п..
> Значит, при $x_n < y_n$ всегда $a \leq b$, что и требовалось доказать.
> 
> Теорема также справедлива для $x_n \leq y_n$ #доказать

*Замечание*. При переходе к пределу в неравенстве $x_n < y_n$ мы всё ещё можем получить тождество $\lim\limits_{n\to\infty}{x_n} = \lim\limits_{n\to\infty}{y_n}$.

> **Теорема (7)**.
> 
> Если ч.п. $\{x_n\}$ имеет своим пределом число $a > p$, то $\exists N: \forall n > N: x_n > p$
> 
> **Доказательство**: выберем произвольное $\varepsilon$, удовлетворяющее неравенству $0 < \varepsilon < a - p$. По данному $\varepsilon$ найдется такой номер $N(\varepsilon)$, что $\forall n > N(\varepsilon)$: $|x_n - a| < \varepsilon$, то есть $a - \varepsilon < x_n < a + \varepsilon$. Учитывая, что $a - p > \varepsilon \implies a - \varepsilon > p$, $x_n > a - \varepsilon$ получим, что $x_n > a - \varepsilon > p \implies x_n > p$, что и требовалось доказать.


> **Теорема (8)**
> 
> Сходящаяся последовательность ограничена.
> 
> **Доказательство**
>  
> Пусть $x_n \to a$. Тогда $\forall \varepsilon > 0: \exists N(\varepsilon): \forall n > N: \vert x_n - a \vert < \varepsilon$
> Известно, что $|x_n| - |a| \leq |x_n - a|$ и, следовательно, $|x_n| - |a| < \varepsilon$, или  $x_n < |a| + \varepsilon$.
> Пусть $M = \max\{|x_1|, |x_2|,\dots,|x_n|, |a| + \varepsilon\}$, тогда $\forall n: |x_n| \leq M$, то есть $M$ ограничивает $\{x_n\}$, что и требовалось доказать.

> **Теорема (10)**
> 
>  Предел суммы равен сумме пределов:
> $$\lim\limits_{n\to\infty}{(x_n + y_n)} = \lim\limits_{n\to\infty}{x_n} + \lim\limits_{n\to\infty}{y_n}$$
> 
> **Доказательство**
> 
> Пусть $\lim\limits_{n\to\infty}{x_n} = a, \lim\limits_{n\to\infty}{y_n} = b, x_n = a + \alpha_n, y_n = b + \beta_n$, где $\alpha_n \to 0, \beta_n \to 0$ при $n \to \infty$.
> Тогда $x_n + y_n = (a + b) + (\alpha_n + \beta_n)$
> Т.к. $\{\alpha_n + \beta_n\}$ — б.м. ч.п., то $x_n + y_n \to a + b$ при $n \to \infty$, что и требовалось доказать.

> **Теорема (11)**
> 
> Предел разности равен разности пределов:
> $$\lim\limits_{n\to\infty}{(x_n - y_n)} = \lim\limits_{n\to\infty}{x_n} - \lim\limits_{n\to\infty}{y_n}$$
> 
> **Доказательство**
> 
> Пусть $\lim\limits_{n\to\infty}{x_n} = a, \lim\limits_{n\to\infty}{y_n} = b, x_n = a + \alpha_n, y_n = b + \beta_n$, где $\alpha_n \to 0, \beta_n \to 0$ при $n \to \infty$.
> Тогда $x_n - y_n = (a - b) + (\alpha_n - \beta_n)$.
> Т.к. $\{a_n - \beta_n\}$ — б.м. ч.п., то $x_n - y_n \to a - b$ при $n \to \infty$, что и требовалось доказать.
> 

> **Теорема (12)**
> 
> Предел произведения равен произведению пределов:
> $$\lim\limits_{n\to\infty}{(x_n  y_n)} = \lim\limits_{n\to\infty}{x_n} \lim\limits_{n\to\infty}{y_n}$$
> 
> **Доказательство**
> 
> Пусть $\lim\limits_{n\to\infty}{x_n} = a, \lim\limits_{n\to\infty}{y_n} = b, x_n = a + \alpha_n, y_n = b + \beta_n$, где $\alpha_n \to 0, \beta_n \to 0$ при $n \to \infty$.
> Тогда $x_ny_n = ab + a\beta_n + \alpha_nb + \alpha_n\beta_n$.
> Т.к. $\{a\beta_n + \alpha_nb + \alpha_n\beta_n\}$ — б.м. ч.п., то $x_ny_n \to ab$ при $n \to \infty$, что и требовалось доказать.
> 
> *Следствие 1*: постоянный множитель можно выносить за знак предела: $$\lim\limits_{n\to\infty}(Cx_n) = C\lim\limits_{n\to\infty}x_n$$
> *Следствие 2*: предел целой полож. степени равен степени ее предела $$\lim\limits_{n\to\infty}({x_n})^k = (\lim\limits_{n\to\infty}{x_n})^k$$

> **Теорема (13)**
> 
> Предел частного равен частному пределов:
> $$\lim\limits_{n\to\infty}{\frac{x_n}{y_n}} = \frac{\lim\limits_{n\to\infty}{x_n}}{\lim\limits_{n\to\infty}{y_n}}$$
