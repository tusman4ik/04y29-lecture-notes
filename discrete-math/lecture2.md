# Sets: Basic Consepts

> [!Размер множеств]
> Обозначение: |A|
> равняется количеству элементов в множестве
> Другое название Кардинальность(почти)

>[!Подмножества]
>$A\subseteq B$ - нестрогое подмножество
>$A \subset B$ -  строгое подмножество

Некоторые примеры подмножеств:
$A\subseteq A$ - true
$\varnothing \subseteq A$ - true

Круги Эйлера и Диограмма Венна
Они есть

> [!Непересекающиеся множества]
>$A \cap B = \varnothing$

> [!Попарно непересекающаяся группа множеств]
>$A \cap B = \varnothing$ для всех пар группы

> [!Разбиение множества]
>n непересикающихся множеств B объединение которых = A
>$B \neq \varnothing$

Как проверить является ли коллекция множеств разбиением А?
1. Все подмножества не пустые
2. Все пары подмножеств не пересекаются
3. Пересечение всех подмножеств = А

# Картежы, Пары

## Картеж(Tuple)
>[!Картеж]
>Картеж это упорядоченная коллекция элементов, которые могут повторяться
>Как массив в проге

> [!Равенство кортежей]
> Два картежа равны если равны их длины, и $a_{i} = b_{i}$ для всех i внутри картежа

A = (1, 2) - кортеж
B = (1,) - кортеж, запятая обязательна

> [!Упорядоченная пара]
> Это кортеж размера 2
> (a, b) = {{a}, {a, b}} - перевод в множества

Как превратить кортеж в множество?
(1, 2, 3) = (1, (2, (3, $\varnothing$))) - пара в паре в паре, мы уже умеем превращать пары в множества

Это только один из способов, можно например откусывать последний элемент

### Декартово произведение картежей
>[!Умножение]
>$A\times B = \{(a,b)|a\in A\ and\ b \in B\}$

>[!Умножение]
>$A^n = A \times A \times A \dots$
